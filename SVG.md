### SVG

The following is a list of supported attributes: 
```
"id", "inkscape:label", "fill", "stroke",
"stroke-width", "stroke-linecap", "stroke-linejoin",
"transform", "clip-path", "style"
```

The following is a list of supported elements:
```
<defs>
  <clipPath>
    <use/>
  </clipPath>
</defs>
<g>
  <path/>
  <line/>
  <polyline/>
  <polygon/>
  <ellipse/>
  <circle/>
  <rect/>
</g>
```
